# Lab Assignment 2

### Commands

Use this command to run the code </br>
`clang++ -Xpreprocessor -fopenmp main_omp.c lab2_io.c lab2_omp.cpp -lomp -o pca`</br>

`./pca <input filename> <retention>`</br>

Example </br>

`./pca testcase_1000_1000 90`
